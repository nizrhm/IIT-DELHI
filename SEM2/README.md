## ⚡ Semester Highlights

This semester combined advanced core computer science domains (Security, Networking, Databases, Systems Lab) with professional coursework in Entrepreneurship and Value Education.

* **Systems & Networking:** Automated data collection with active shaping, built kernel-like utilities, and constructed an early flow classifier using CALIMERA & MiniRocket.
* **Security & LLMs:** Focused heavily on security engineering, implementing Wireshark traffic analyzers, cryptographic decoders, and specialized defenses for Generative AI APIs.
* **Databases:** Scaled vector search databases (FAISS) using Product Quantization techniques and developed a custom adaptive reranking pipeline to optimize recall vs latency.
